# fish-enterprise

- Фронтенд react native + expo
- База данных neo4j
- API GraphQL
- Apollo GraphQL Server

- реализована подгрузка данных страницами при нажатии кнопки load more
- фильтрация данных на каждой из страниц через запросы GraphQL
